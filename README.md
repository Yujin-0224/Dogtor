# Dogtor  
AI 기반 반려견 건강 진단 모바일 애플리케이션

## 프로젝트 소개
Dogtor는 반려견의 눈병과 피부병을 AI로 분석하여
보호자가 반려견의 건강 상태를 쉽게 확인할 수 있도록 돕는 모바일 앱입니다.  
사진 기반 AI 진단, AI 챗봇 상담, 근처 동물병원 안내 기능을 제공하여  
반려견의 상태 확인부터 상담, 병원 방문까지 하나의 흐름으로 지원하는 것을 목표로 합니다.

## 개발 목적
- 반려견의 눈병 및 피부병을 조기에 발견
- 병원 방문 전 보호자가 기본적인 건강 상태를 확인할 수 있도록 지원
- AI 기술을 활용한 직관적이고 접근성 높은 반려견 건강 관리 서비스 제공

## 주요 기능

### 1. AI 진단 기능
사용자가 반려견의 눈 또는 피부 사진을 업로드하면  
AI가 이미지를 분석하여 질병 여부를 분류합니다.  
진단 결과에는 질병 이름과 함께 예측 신뢰도가 표시됩니다.

- 눈병 진단
- 피부병 진단
- 사진 촬영 가이드 제공으로 진단 정확도 향상
- 분석 중 로딩 UI 제공

### 2. AI 챗봇 상담
OpenAI GPT 모델을 활용한 챗봇 기능입니다.  
사용자는 반려견의 건강, 행동, 관리 방법 등에 대해 질문할 수 있으며  
AI Dogtor가 친근한 가상 수의사 보조 역할로 상담을 제공합니다.

- 자연스러운 대화 인터페이스
- 반려견 건강 관련 질문 응답
- AI 진단 결과에 대한 추가 설명 가능

### 3. 동물병원 위치 안내
Google Maps API를 이용하여  
사용자의 현재 위치를 기준으로 근처 동물병원을 지도 위에 표시합니다.

- 주변 동물병원 검색
- 병원 위치, 주소, 전화번호 정보 제공
- 지도 기반 직관적인 UI

## 사용 기술 및 API
- Android (Jetpack Compose)
- Roboflow API (이미지 기반 AI 진단)
- OpenAI GPT API (챗봇 기능)
- Google Maps API, Places API (동물병원 위치 안내)

## AI 모델 정보

### 눈병 진단 모델
- 데이터셋: Dog Eye Problems Detection (Roboflow)
- 데이터 수: 2,652장
- 클래스 수: 4개
  - 정상
  - 결막염
  - 안검내반
  - 눈꺼풀 혹
- 데이터 분할 비율: 학습 70퍼센트 / 검증 20퍼센트 / 테스트 10퍼센트
- 평균 정밀도: 91.7퍼센트
- 모델 유형: 사전 학습된 이미지 분류 모델

### 피부병 진단 모델
- 데이터셋: dog-skin-disease-dataset (Roboflow)
- 데이터 수: 1,418장
- 클래스 수: 4개
  - 정상
  - 세균성 피부염
  - 곰팡이 감염
  - 알레르기
- 평균 정밀도: 97.7퍼센트
- 모델 유형: 사전 학습된 이미지 분류 모델

## 데이터 전처리 방식
Roboflow 플랫폼에서 제공하는 전처리 파이프라인을 사용하였습니다.

- 이미지 크기 정규화
- 데이터 증강 (회전, 반전 등)
- 클래스 라벨 정제
- 학습, 검증, 테스트 데이터 자동 분할

## UI 설계 개요
Dogtor 앱은 다음 네 가지 주요 화면으로 구성됩니다.

1. 메인 화면  
   주요 기능에 쉽게 접근할 수 있도록 구성된 홈 화면으로  
   따뜻한 색감과 마스코트를 활용해 친근한 사용자 경험을 제공합니다.

2. AI 검사 화면  
   사진 업로드를 통해 AI 진단을 수행하는 화면으로  
   촬영 가이드와 분석 결과를 함께 제공합니다.

3. 챗봇 화면  
   AI Dogtor와의 대화를 통해 반려견 건강 상담을 진행하는 화면입니다.

4. 지도 화면  
   사용자의 위치를 기반으로 근처 동물병원을 지도 위에 표시하는 화면입니다.

## 유의사항
본 앱에서 제공하는 AI 진단 결과는 참고용이며  
실제 수의사의 전문적인 진단을 대체할 수 없습니다.  
이상 증상이 지속될 경우 반드시 동물병원을 방문해야 합니다.
