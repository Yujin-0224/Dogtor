Dogtor
AI 기반 반려견 건강 진단 모바일 앱

프로젝트 소개
Dogtor는 반려견의 눈병과 피부병을 AI로 진단하고,
보호자가 반려견의 건강 상태를 쉽게 확인할 수 있도록 돕는 모바일 애플리케이션입니다.
사진 기반 AI 진단, 챗봇 상담, 근처 동물병원 안내 기능을 제공합니다.

개발 목적
- 반려견의 건강 문제를 조기에 발견
- 병원 방문 전 보호자가 기본적인 상태를 파악할 수 있도록 지원
- AI를 활용한 간편한 반려견 건강 관리 서비스 제공

주요 기능
1. AI 건강 진단
사용자가 반려견의 눈 또는 피부 사진을 업로드하면
AI가 이미지를 분석하여 질병 여부를 판단합니다.
진단 결과와 함께 신뢰도를 제공합니다.

2. AI 챗봇 상담
OpenAI GPT 모델을 활용한 챗봇 기능으로
반려견의 건강, 행동, 관리 방법에 대한 질문에
자연스러운 대화 형식으로 답변을 제공합니다.

3. 동물병원 위치 안내
Google Maps API를 이용하여
사용자의 현재 위치를 기준으로
가까운 동물병원을 지도 위에 표시합니다.
병원의 위치, 주소, 전화번호 정보를 확인할 수 있습니다.

사용 기술 및 API
- Roboflow API: 반려견 눈병 및 피부병 이미지 진단
- OpenAI GPT API: AI 챗봇 기능 구현
- Google Maps API, Places API: 동물병원 위치 안내
- Android, Jetpack Compose

AI 모델 정보

눈병 진단 모델
- 데이터셋: Dog Eye Problems Detection (Roboflow)
- 클래스: 정상, 결막염, 안검내반, 눈꺼풀 혹
- 데이터 수: 2,652장
- 평균 정밀도: 91.7퍼센트

피부병 진단 모델
- 데이터셋: dog-skin-disease-dataset (Roboflow)
- 클래스: 정상, 세균성 피부염, 곰팡이 감염, 알레르기
- 데이터 수: 1,418장
- 평균 정밀도: 97.7퍼센트

유의사항
본 앱의 AI 진단 결과는 참고용이며
실제 수의사의 진단을 대체할 수 없습니다.
이상 증상이 있을 경우 반드시 동물병원을 방문해야 합니다.
